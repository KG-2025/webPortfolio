PWA for McDonald's Franchise

Overview

TuPlanRedondo is a Progressive Web App (PWA) developed for a McDonald's franchise to streamline the process of employee evaluations and data collection.
The app serves as a tool for collecting and analyzing performance data, offering management real-time insights through an integrated dashboard. The franchise needed a user-friendly solution that didn’t require app downloads and ensured secure access through user credentials. The system was built to work seamlessly across tablets, smartphones, and laptops, offering flexibility and convenience for both field employees and management.

The Problem

The franchise was managing employee performance evaluations manually, which was time-consuming and prone to data errors. There was no centralized digital tool in place to gather, track, and analyze this data effectively. Additionally, the client requested a lightweight solution that didn’t require the users (employees or managers) to download an app, yet still provided robust functionality for both data collection and reporting.

Hypothesis

Developing a PWA would offer a flexible and cost-effective solution, eliminating the need for native app development while ensuring accessibility across all devices through a web browser.
TThe PWA would ensure quick data collection, real-time analytics, and ease of deployment. It was expected that this approach would reduce manual errors, improve data accuracy, and give management better insights into performance trends.

Role & Responsibilities

  • User Research: Desk Research, 5 User interview, Affinity diagram, Personas, User journey, 10 Usability Test to define the key requirements and ensure the app was tailored to the client’s specific needs.
  • User Interface Design: created wireframes and mockups focused on simplicity and usability, ensuring the app would function efficiently on tablets, mobile phones, and laptops..
  • Graphic Design: created all icons and visual elements to match the McDonald’s branding guidelines while maintaining a clean and functional look.
  • Front-End Development: development of the PWA, ensuring it was lightweight, responsive, and functional across all devices without the need for installation.

Technologies

Figma -HTML5 - Bootstrap 4 - CSS3 - JavaScript - PWA

Output

  • Data Collection Efficiency: the app streamlined the data collection process, reducing the time spent on manual evaluations by 30%, and allowing managers to quickly input data from any device.
  • Real-Time Reporting the dashboard feature provided managers with real-time insights into employee performance, resulting in more informed decision-making and faster responses to issues.
  • Cost Savings: the use of a PWA eliminated the need for the franchise to invest in a native app, reducing development costs by approximately 30%.
  • User Satisfaction: the simplicity and accessibility of the app led to a 25% increase in user satisfaction, as employees and management appreciated the ease of use and quick access to data.

This project provided a scalable, user-friendly solution for the franchise, significantly improving the efficiency of their evaluation process while offering valuable insights through a real-time dashboard.
